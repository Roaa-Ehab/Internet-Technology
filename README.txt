MUST Connected Dashboard
========================

This package connects the admin dashboard to website/index.html.

How to run:
1. Open a terminal inside this folder.
2. Run: python server.py
3. Open the dashboard at: http://127.0.0.1:8000/admin-dashboard/admin-dashboard.html
4. Open the website at:   http://127.0.0.1:8000/website/index.html

Dashboard password:
- mustadmin2026

What it does:
- Add and remove news items.
- Add and remove upcoming events.
- Save changes directly into website/index.html.

Important:
- Keep the server running while you use the dashboard.
- The dashboard edits the managed News and Upcoming Events content blocks inside website/index.html.
